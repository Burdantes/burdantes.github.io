What's behind the numbers

Measurements come from M-Lab: NDT speed tests run by real users on school networks, annotated with per-hop routing detail. We filter each dataset to known school IP ranges, reconstruct the forward path to domestic and international test servers, and summarize path length, exchange-point crossings, transit-country dependency, latency decomposition, loss, and performance over time. Every figure on a country page comes from that pipeline.
